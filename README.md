# imooc-admin

## 描述

这个项目改写于vue-element-admin

> 如果这个项目能够帮助到你，请记得star哦！

功能：
    - 基于i18n实现国际化
    - 基于scss实现的动态换肤
    - 实现了excel导入导出
    - 基于wangEditor和tui.editor实现了富文本和markdown编辑功能
    - 基于sortablejs实现的拖拽排序
    - 基于vue-print-nb实现的打印功能

## 项目运行

    - 安装依赖
        ```
        npm install
        ```
    - 启动项目
        ```
        npm run serve
        ```
    - 打包项目
        ```
        npm run build
        ```
    - 发布项目
        ```
        npm run release
        ```
